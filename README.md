# Stopwatch Timer App

Minimal desktop stopwatch + timer built in Python.

## Run

python3 src/stopwatch-timer-app.py

## Features

- Stopwatch  
- Timer  
- Minimal UI
- Light/Dark Theme switcher
- Roller Picker (iOS Style)
- Expandable architecture
